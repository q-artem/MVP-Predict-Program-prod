# Добро пожаловать в MVP Predict Program

## Система интеллектуальной аналитики успеваемости студентов

Это приложение предназначено для выявления студентов в зоне академического риска и формирования прогнозов успеваемости.

*Инструкции по использованию см. в пользовательском README*

### Локальный запуск:
0. Склонировать репозиторий `git clone https://github.com/q-artem/MVP-Predict-Program-prod.git`
1. Перейти в папку `cd MVP-Predict-Program`
2. Создать виртуальное окружение `uv venv .venv`
3. Активировать виртуальное окружение `source .venv/bin/activate.fish`
4. Перейти в папку с исходниками `cd src`
5. Установить библиотеки `uv pip install -r requirements.txt`
6. Запустить `streamlit run app.py`
7. Перейти по ссылке из консоли
