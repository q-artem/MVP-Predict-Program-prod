# Добро пожаловать в MVP Predict Program

## Система интеллектуальной аналитики успеваемости студентов

Это приложение предназначено для выявления студентов в зоне академического риска и формирования прогнозов успеваемости.

### Локальный запуск:
0. Склонировать репозиторий `git clone https://github.com/q-artem/MVP-Predict-Program-prod.git`
1. Перейти в папку `cd MVP-Predict-Program`
2. Создать виртуальное окружение `uv venv .venv`
3. Активировать виртуальное окружение `source .venv/bin/activate.fish`
4. Перейти в папку с исходниками `cd src`
5. Установить библиотеки `uv pip install -r requirements.txt`
6. Запустить `streamlit run app.py`
7. Перейти по ссылке из консоли

## Использование

**Быстрый старт:**
1. Перейдите на вкладку **«Загрузка данных»**
2. Загрузите файл с текущими показателями студентов (в формате `.csv`)
3. Перейдите на вкладку **«Прогнозирование»**, выберите тип задачи (Экзамен или Итоговая оценка) и получите отчет

👈 *Используйте боковое меню для навигации.*

### Вкладка "Прогнозирование"
Вкладка предназначена для получения предсказания оценок студентов
#### Индивидуальный профиль
По выбору студента из загруженного файла представляет: 
- Краткую общую информацию о студенте
- Предсказание его оценки за конкретный предмет и семестр (по умолчанию последний)
- Интерпретацию прогноза (какие признаки оказали наибольшее положительное и отрицательное влияние на итоговый прогноз)
  - Левый столбец -- факторы, упорядоченные по уменьшению влияния сверху вних
  - Правая область -- Влияние факторов
    - Влияние суммируется снизу вверх
    - Значение E[f(x)] -- среднее значение, на которое повлияли описанные факторы
    - Далее снизу вверх факторы прибавляют (красная стрелка вправо) или вычитают (синяя стрелка влево) своё значение из среднего
    - Значение f(x) вверху -- предсказание модели по данному студенту
  - График позволяет визуально выделить факторы с наибольшим влиянием на оценку студента
- Сравнение оценки студента со средней по потоку
- Профиль компетенций студента в сравнении со средними показателями по вузу
#### Массовый расчёт
Вкладка позволяет предсказать оценку для всех студентов из загруженного файла и экспортировать результат в файл `.csv`
#### Ручной ввод?
Вкладка позволяет вручную ввести данные для получения предсказания по "абстрактному" студенту. (В разработке)

Может использоваться для получения прогноза по студентам, отсутствующим в базе или для проверки гипотезы (например, 'что будет, если GPA станет 4.5')

### Вкладка "Глобальная аналитика"
Вкладка предназначена для просмотра интерактивных графиков, построенных на основе введённых данных

#### Работа с графиками
- Все графики интерактивные, то есть можно получать дополнительную информацию при наведении курсора на элементы графика
- Все графики можно развернуть на весь экран
  - Чтобы развернуть график, нужно нажать на кнопку "Fullscreen" в правом верхнем углу графика
  - Чтобы свернуть график обратно -- Esc или нажать на ту же кнопку
- У всех графиков можно сделать снимок, т. е. сохранить статическое изображение данного графика
  - Для этого нужно нажать на иконку фотоаппарата в правом верхнем углу графика
    - Если сделать это в полноэкранном режиме, сохранённое изображение получится более детальным
- У графиков с диаграммами есть возможность приближения определённой области графика.
  - Это можно делать навигационными кнопками в правом верхнем углу графика
  - Или выделив интересующую область графика зажатием мыши. Чтобы вернуть исходный масштаб -- сделать двойной клик в любом месте графика
    - Например, можно выделить интересующий семестр на графике "Индекс академической нагрузки" или плотную область на графике "Рейтинг сложности дисциплин"
    - При сохранении снимка приближенного графика на снимке будет отображена та же область

## Обучение модели
### ПЕРВИЧНОЕ ОБУЧЕНИЕ (train)
(когда нет моделей, или хотим, чтобы с 0 обучить модель)

`python app.py train arg1 arg2 input.csv output.csv "arg3"`

`arg1` — 0 если хотим бинарную модель, 1 если качественную модель

`arg2` — 1 если хотим обучить на всех данных, Х на каком-то количестве данных (если Х = 0.8, значит 80% будет для обучения, 20% отсеется и выйдет в output.csv

`arg3` — это название целевого столбца (target), то есть колонка, которую модель должна предсказывать.
Примеры запросов:
1) `python app.py train 0 0.6 data/table_NaN_binary.csv binary_test.csv "Оценка (0 или 1)"`
2) `python app.py train 1 1.0 data/table_NaN_quality.csv quality_test.csv "ОЦЕНКА_NUM"`
### ДООБУЧЕНИЕ (retrain)
(когда модели есть и хотим их улучшить новыми данными. Данные поглощаются полностью)

`python app.py retrain arg1 input.csv “arg3”`

Пример запросов:
1) `python app.py retrain 0 new_binary.csv "Оценка (0 или 1)"`
2) `python app.py retrain 1 new_quality.csv "ОЦЕНКА_NUM"`

### ПРЕДСКАЗАНИЕ (predict) 
(когда модели есть и хотим предсказания, занимает очень много времени из-за SHAP и вывода факторов влияния, у меня это по 30 минут каждый, можно спокойно идти обедать)

`python app.py predict arg1 input.csv output.csv`
Пример запросов:
1) `python app.py predict 0 binary_test.csv binary_output.csv`
2) `python app.py predict 1 quality_test.csv quality_output.csv`
